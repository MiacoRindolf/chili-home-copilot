## Sequencing recommendation (post-jumbo)

The high-impact work from the original sequencing all shipped during `f-overnight-jumbo-2026-05-06`. Remaining backlog is medium-low impact:

1. **`f-fix-autotrader-paper-fallback`** â€” operator design decision pending; live and paper are mutually-exclusive modes today, so a true fallback requires an architectural choice rather than a code patch.
