### money/amount.py
```python
from dataclasses import dataclass


@dataclass(frozen=True)
class Money:
    currency: str
    amount: float

    @classmethod
    def from_value(cls, currency, value):
        return cls(currency, float(value))

    def formatted(self):
        return str(self.amount)
```
